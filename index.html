Task

A Narcissistic Number is a positive number which is the sum of its own digits, each raised to the power of the number of digits in a given base. Your code must return true or false depending upon whether the given number is a Narcissistic number in base 10. 
function narcissistic(value) {
    let list = String(value).split("");
    let summ = list.reduce(function(previousValue, currentValue){
        return previousValue + currentValue ** list.length
    }, 0);
    return summ === value; 
}
